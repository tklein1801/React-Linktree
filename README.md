## :page_facing_up: About

You are free to use this Website but you need replace the image mine(located here: `arc/assets/img/me.jpg` & `public/me.jpg`) with your own images

## :rocket: Features

- Responsive Design
- Get links from .JSON-file located in `src/assets/data/links.json`

## :wrench: Installation

**1.** Install the required dependencies using `npm install`
**2.** Start an server with `npm start`
